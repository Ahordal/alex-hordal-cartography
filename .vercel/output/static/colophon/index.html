On The Front Cover

The image on the front page of this website is a grayscale representation of a map I created titled Hawaiʻi:Seafloor to Summit. The image shows the full vertical extent—from seafloor to summit—of a portion of the eight main Hawaiian Islands represented using a topobathymetric DEM and hillshading.

Tech Stack

The website itself is built using Astro, MDX, Vercel, GitHub, and Font Awesome, with a strong emphasis on semantic and validated HTML, modern CSS architecture, performant image delivery, structured long-form project documentation, and scalable content architecture for cartographic and GIS-focused work.

AI-Assisted Development

This website was developed through a combination of traditional frontend development and iterative AI-assisted workflows. Generative AI was used throughout the project as a conversational tool for troubleshooting, layout refinement, component exploration, and development support. The site architecture, content organization, and overall editorial direction were developed incrementally through ongoing experimentation and refinement.
